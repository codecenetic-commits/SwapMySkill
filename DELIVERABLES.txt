## URLs
- Main Landing Page: http://localhost:3000/
- Design System: http://localhost:3000/design

## How to Run
1. Install dependencies: npm install
2. Run development server: npm run dev
3. Build for production: npm run build

## Features Implemented
✓ Responsive Design (Mobile → Tablet → Desktop)
✓ Immersive Hero with 3D Visualization
✓ Smooth Animations (Framer Motion + GSAP)
✓ Smooth Scrolling (Lenis)
✓ Dark/Light Theme Toggle
✓ Accessibility Features
✓ Performance Optimizations
✓ All Required Sections (Navigation, Hero, How It Works, etc.)
✓ Design System with Component Showcase
✓ Comprehensive Documentation
